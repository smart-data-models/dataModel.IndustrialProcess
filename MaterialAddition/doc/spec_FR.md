<!-- 40-NotesYaml -->  
Ce modèle de données fournit des données relatives aux ajouts de matériaux effectués dans une usine de process. Le modèle de données a été développé à partir des besoins de la sidérurgie, mais des structures de données similaires devraient également être appropriées dans d'autres domaines d'application. Le processus peut être discontinu (batch) ou continu.
